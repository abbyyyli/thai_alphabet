Overview｜泰文拼讀的核心流程

看到一個泰文字時，先不要急著猜

你真正要練的是固定流程：

  1. 找主子音。
  2. 找母音形狀。
  3. 找尾音。
  4. 判斷短音 / 長音 / 複合母音。
  5. 先組合出基礎讀音。
  6. 最後再處理聲調。
本篇先練「拼得出來」，Part 4 再專門處理完整聲調規則。

示範：เรื่อง

ร + เ-ือ + ง → เรื่อง rʉ̂ang matter; topic

เ-ือ 是 ʉa，尾音 ง 是 -ng，組合後是 rʉang，再加上聲調就是 rʉ̂ang。

本篇最重要目標

BeforeAfter

Reading Formula｜拼讀公式

Formula 1子音 + 母音

ม+า→มา maacome

子音和母音本身都可點擊播放；最後再聽完整字。

Formula 2子音 + 母音 + 尾音

บ+า+น→บ้าน bâanhouse

尾音也可以單獨播放，幫你練習收尾。

Formula 3前置母音：母音寫前面，但發音仍在子音後

เ◌+ก→เก geelong ee pattern

使用 dotted circle 顯示母音位置，避免手機瀏覽器把橫線顯示錯。

Formula 4複合母音 + 尾音

ม+เ◌ือ+ง→เมือง mʉangcity

複合母音要整組看，不要把符號拆散。

Leading Consonants｜前引字

前引字怎麼影響聲調？

前引字不是單純「把後面變高音字」。更精準地說：前面的子音會把自己的子音類別帶進聲調判斷，影響後面的低音子音該用哪一套聲調規則。

  • 高音前引：例如 ขนม / สนุก，前面的高音字母影響下一個低音子音，常見結果是第一音節帶短 a，後一音節照被前引後的規則判斷。
  • 中音前引：例如 ตลาด，前面的中音字母也可能形成前引效果，後面音節的聲調判斷不能只看後面那個低音子音。
  • ห นำ：最規則、最常見。ห 幾乎不發音，但讓後面的低音子音照高音子音規則判斷聲調。
  • อ นำ：主要記 อย่า / อยู่ / อย่าง / อยาก，อ 本身不照一般發音，整組讀成 y- 起音。
  • 實務上先認高頻字，比硬背抽象規則更有效；聲調細則會和 Part 4 連動。
