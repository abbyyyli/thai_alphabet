廣告體 Advertising Script

這份不是字體設計課，而是「看到泰國招牌、菜單、IG 圖卡時可以快速對照」的辨認筆記。以 Kanit 作為清楚範例，整理廣告體 / 圓體 / 粗體 / 仿手寫體中最容易看錯的泰文字形。

什麼是數位廣告體？

泰國現代招牌多用電腦輸出印刷，選用的字型以 Kanit、Prompt 為主流。這類字型屬於幾何窄身無襯線體——比教科書常用的 Noto Sans Thai 明顯更窄、更粗，字形比例也不同。

初學者習慣了 Noto Sans Thai 的「工整版」泰文後，看到 Kanit 風格的招牌會感到陌生——不是認不得字母，而是比例感不對、字間距極緊、粗細對比強，需要重新校正視覺直覺。

Kanit 的三個閱讀重點

①字形更窄、字間更緊

Kanit 是 condensed 字型，每個字母比 Noto Sans Thai 窄約 15–20%。招牌上常把字間距再壓緊，整個詞看起來像一個視覺塊。

อาหาร
Noto — 較寬鬆
อาหาร
Kanit — 更窄更緊

②字頭圓圈更小、更俐落

Kanit 的幾何風格讓字母頭部（หัว）比 Noto Sans Thai 更小、更緊湊，習慣標準體的人需要重新抓字母特徵的位置感。

กขคงจ
Kanit — 字頭更小更緊

③上下母音符號位置改變

Kanit 的行高較緊，字母上方的母音符號（อิ อี อึ อื）和聲調符號離子音更近，視覺上不如 Noto 分明，需要適應。

Noto — 符號清晰
กิ กี กึ กื
Kanit — 符號更緊湊

④字重越重、字形越方

招牌常用 700–800 weight。越粗的 Kanit，筆畫間的空隙越小，字形越接近方塊。看到非常粗的泰文字，先從整體詞形抓輪廓，再辨認個別字母。

Kanit 閱讀規則

從 Noto Sans Thai 切換到 Kanit 時，需要重新校正的視覺習慣。

規則 1整體字形更窄——不要誤判字母

Kanit 是 condensed 字型，每個字母水平方向壓縮約 15–20%。最容易出錯的是原本「較寬」的字母，在 Kanit 裡看起來像另一個字。

字母Rom.Kanit 下的視覺陷阱
ณ n 壓縮後內部空間縮小，容易看成 ม 或 น
ฆ kh 兩側弧形因壓縮而更靠近，需留意上方橫線
ถ / ภ th / ph 原本就相似，Kanit 下更難分，靠左側筆畫

規則 2字頭（หัว）更小、更規整

Kanit 的幾何風格使字頭（起筆小圓圈）統一縮小且位置更固定，反而比 Noto Sans Thai 的字頭更一致易找——只是要適應它「在更靠近字母主幹」的位置。

กขคงจชนบปมยรลวสหอ
全部子音的 Kanit 字頭一覽（weight 700）
💡 Kanit 字頭的位置和大小比 Noto 更統一，辨認時抓住「字頭在哪側」（左上 / 右上）就能快速分類。

規則 3粗重字重下，相似字母更難區分

招牌常用 weight 700–800，筆畫變粗後，字母間的空隙縮小，以下幾組特別容易混淆：

易混組Rom.區分方法（Kanit 700）
ด ต d / dt ด 的右上有一個小外突，ต 無——在 Kanit 700 下仍可見，但很細微
บ ป bp / bp บ 左上有小圓頭，ป 的上橫更長且無圓——重量越大越難分，靠詞義
ภ ถ ph / th ถ 左側有一直線延伸向上，ภ 左側為弧形——找左邊的形狀
น ม n / m น 整體較窄，右側弧較高；ม 較寬方，Kanit 下仍有明顯差異

規則 4母音符號讀法不變，位置感要重建

Kanit 的行高較緊，上方母音（อิ อี อึ อื อั）和聲調符號距子音更近，不像 Noto 那樣有清楚呼吸感。母音本身沒有改變，只是視覺上更擁擠。

กิ กี กึ กื กั
Noto — 符號空間充裕
กิ กี กึ กื กั
Kanit — 符號更緊

規則 5前置母音（เ แ โ ไ ใ）在 Kanit 的閱讀順序

前置母音放在子音左邊，在 Kanit 窄身排列下字間距更緊，視覺上容易把前置母音誤讀為上一個字的一部分。

💡 提醒：看到 เ แ โ ไ ใ 時，它一定是緊跟在右邊那個子音的母音，不是左邊那個字的尾巴。這在 Kanit 緊排版下需要特別意識。
เปิด
เ 屬於 ป，不是前一個字
ไม่ใช่
ไ 屬於 ม，ใ 屬於 ช

字母對照｜廣告體快速辨認筆記

這一頁不是要分辨每一種字體名稱，而是訓練你看到泰國招牌、菜單、IG 圖卡時，快速判斷「這到底是哪個字母」。先看 Level A：最容易看錯、最值得背辨認點的組合。

Level A｜最容易看錯，優先練

บvsป
Level A
บ / b, ป / bp（但你系統常用 bp 對 ป；兩者 class 不同）
快速判斷：บ 找左上小圓頭；ป 找上方長橫、沒有圓頭。
  • บ 是低音字；ป 是中音字，聲調規則不同。
  • 粗體時圓頭會縮小，先看「上方是否有長橫」。
ผvsพ
Level A
ผ / ph（高音字）, พ / ph（低音字）
快速判斷：ผ 左側比較像立起來的開口；พ 更像兩個圓弧連在一起。
  • 兩個都讀 ph，但 class 不同，聲調會不同。
  • 廣告體變胖時，先看左邊是不是比較直、比較開。
ถvsภ
Level A
ถ / th（高音字）, ภ / ph（低音字）
快速判斷：ถ 左邊有直線往上；ภ 左邊是弧形、像包起來。
  • 這組在粗體招牌很容易看錯。
  • 先看左側：直線 = ถ；圓弧 = ภ。
ยvsญ
Level A
ย / y, ญ / y 或 final -n
快速判斷：ญ 底部多一個下伸／尾巴；ย 比較簡短、沒有複雜下尾。
  • 手寫或廣告體常把 ญ 的尾巴縮短。
  • 看到女性 หญิง / yǐng 時，常會遇到 ญ。
รvsว
Level A
ร / r, ว / w
快速判斷：ร 比較像短鉤；ว 比較像開口圓弧。
  • 廣告體中 ร 會變很短，很容易被略過。
  • ว 常出現在 วัน / วาง / ขาว / เร็ว 這類詞。
ไvsใ
Level A
ไ / ai, ใ / ai（同音）
快速判斷：ใ 上方鉤往左；ไ 上方鉤往右。
  • 發音相同，但拼字不同。
  • ใ 只出現在固定 20 個詞；看到 ใจ / ใช่ / ใบ 要特別記形狀。

Level B｜也很常見，看到會卡一下

เvsแ
Level B
เ / ee family, แ / ɛɛ family
快速判斷：แ 是雙線；เ 是單線。廣告體緊排時，แ 的兩條線可能黏得很近。
  • 看到 แพง / แม่ / แมว 時，要意識到是 แ，不是 เ。
  • 前置母音永遠屬於右邊的子音。
าvsำ
Level B
า / aa, ำ / am
快速判斷：ำ = 上面小圈 + า。粗體時小圈可能很貼近主字。
  • ราคา 是 า；กำลัง 是 ำ。
  • 看到上方小圈不要漏掉，漏掉就會把 am 看成 aa。
อvsฮ
Level B
อ / carrier or ɔɔ, ฮ / h
快速判斷：ฮ 比 อ 多出右側／上方的裝飾感，整體更複雜。
  • อ 常當母音承載字或 initial。
  • ฮ 多出現在外來語或口語詞。
นvsม
Level B
น / n, ม / m
快速判斷：ม 比較寬、像有兩段肚子；น 比較窄。
  • Kanit 窄身時兩者都被壓縮，但 ม 通常仍比較寬。
  • 詞尾時靠語感與常見詞判斷。
ชvsซ
Level B
ช / ch, ซ / s
快速判斷：ช 有較明顯的多段弧；ซ 的形狀更像簡化的 s 音字。
  • 兩者在粗體、圓體時會變像。
  • ซ 常見於 ซื้อ / ซอย / ซี 等。
ทvsธ
Level B
ท / th, ธ / th
快速判斷：ธ 有更特殊的內部／上方形狀，ท 較簡單。
  • 兩者同為低音 th，很多時候要靠字形記憶。
  • ธ 常見於 ธนาคาร / ธุระ。

Level C｜補充辨認，遇到再查

組合Romanization快速辨認點高頻例字
ด / ตd / dtด 右上有小外突；ต 較平。兩者都是中音字，聲調邏輯相同。ดี / dii, ต้อง / dtɔ̂ng
ข / ชkh / chข 有上方橫線；ช 是多段弧，沒有橫線。ขาย / khǎai, ชา / chaa
ศ / ส / ษs發音相近，廣告閱讀先認整體詞；拼字細節可回查。สด / sòt, ภาษา / phaa-sǎa
ฤ / ฤๅrʉ / rʉʉ少見但一看到很陌生；先當特殊符號記。ฤดู / rʉ́-duu, ฤๅษี / rʉʉ-sǐi
📌 你的閱讀策略：先不要管這是 Kanit、Prompt、圓體還是手寫體。看到陌生廣告字時，先找「圓頭 / 長橫 / 左側直線 / 底部尾巴 / 前置母音」這幾個辨認點，再用高頻招牌詞反推。
