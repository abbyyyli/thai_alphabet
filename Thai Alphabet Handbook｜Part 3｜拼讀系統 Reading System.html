Overview｜泰文拼讀的核心流程

看到一個泰文字時，先不要急著猜

你真正要練的是固定流程：

  1. 找主子音。
  2. 找母音形狀。
  3. 找尾音。
  4. 判斷短音 / 長音 / 複合母音。
  5. 先組合出基礎讀音。
  6. 最後再處理聲調。
本篇先練「拼得出來」，Part 4 再專門處理完整聲調規則。

示範：เรื่อง

ร + เ-ือ + ง → เรื่อง rʉ̂ang matter; topic

เ-ือ 是 ʉa，尾音 ง 是 -ng，組合後是 rʉang，再加上聲調就是 rʉ̂ang。

本篇最重要目標

BeforeAfter
看到 เพิ่ง 只能靠記憶或查拼音。 知道它是 พ + เ-ิ + ง → phə̂ng。
看到 เปลี่ยน 覺得字母很多。 知道它是 ปล + เ-ีย + น → bplìian。
看到 ห้อง 猜不出 ɔɔ。 知道 อ + ง 這裡讀 ɔɔng → hɔ̂ng。

Reading Formula｜拼讀公式

Formula 1子音 + 母音

ม+า→ มา maacome

這是最直覺的形式：子音在前，母音在後。

Formula 2子音 + 母音 + 尾音

บ+า+น→ บ้าน bâanhouse

有尾音時，尾音會決定收尾方式，例如 น 收 -n。

Formula 3前置母音：母音寫前面，但發音仍在子音後

เ-+ก→ เก geelong ee pattern

泰文很多母音寫在子音前面，但讀音仍然跟在子音後面。

Formula 4複合母音 + 尾音

ม+เ-ือ+ง→ เมือง mʉangcity

複合母音通常最容易讓學習者卡住。先抓整組母音，不要一個符號一個符號硬拆。

Final Reading｜尾音閱讀邏輯

尾音不是看子音原本的 initial sound

同一個子音放在開頭和放在尾巴時，可能讀法不同。Part 2 已經整理過子音，本篇把它轉成「實際閱讀時怎麼收尾」。

Dead finals｜死音節尾音

-k / -t / -p 是之後判斷聲調時很重要的死音節尾音。

Live finals｜活音節尾音

-n / -m / -ng / -y / -w 通常是活音節尾音。

Common Mistakes｜常見拼讀錯誤

1. เ-า / ao 和 อาว / aaw 混在一起

TypeThaiRomanizationEnglish
aoเขา khǎohe; she; they
aawขาว khǎawwhite
aawข้าว khâawrice

2. เ-ือ / ʉa 寫成 rûueang 類型

學習版固定：

不要在同一份筆記中混用 rʉ̂ang 和 rûueang。

3. เลย 不是 loei / looei

本套拼讀手冊使用：

ล+เ-ย→ เลย ləəyso; at all

4. เห็น เอง เงิน เดิน：看到 เ- อย่า一律讀 ee

ThaiPatternRomanizationEnglish

Practice｜練習題

1. 拆字練習

請先自己拆，再看答案。

  1. เรื่อง
  2. เงิน
  3. เปลี่ยน
  4. แล้ว
Answer

เรื่อง / rʉ̂ang / matter = ร + เ-ือ + ง

เงิน / ngoen / money = ง + เ-ิ + น

เปลี่ยน / bplìian / change = ปล + เ-ีย + น

แล้ว / lɛ́ɛw / already = ล + แ- + ว

2. 選正確 Romanization

1. เลย = ?

2. เดิน = ?

3. ห้อง = ?

4. ซื้อ = ?

Answer

เลย / ləəy / so; at all

เดิน / dəən / walk

ห้อง / hɔ̂ng / room

ซื้อ / sʉ́ʉ / buy

3. 看句子讀出來

วันนี้ต้องไปซื้อของ

Answer

วันนี้ต้องไปซื้อของ
wan-níi dtɔ̂ng bpai sʉ́ʉ khɔ̌ɔng
Today I have to go buy things.
