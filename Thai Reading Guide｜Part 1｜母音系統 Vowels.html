泰文音節基本結構

子音 + 母音 + 尾音
initial consonant + vowel + final sound
A Thai syllable is usually built from an initial consonant, a vowel, and sometimes a final sound.

Basic Vowels｜基本母音

1. อะ / อา｜a / aa

短音 a 有尾音時，常變成 อั + final。

無尾音有尾音
กะ / gaกัน / gan / together
จะ / jàจัด / jàt / arrange
ฉัน / chǎn / I
จำ / jam / remember; memorize

2. อิ / อี｜i / ii

短音長音
กิ / giกี / gii
คิด / khít / thinkมี / mii / have; there is

3. อึ / อือ｜ʉ / ʉʉ

無尾音有尾音
มือ / mʉʉ / handคืน / khʉʉn / night; return
ชื่อ / chʉ̂ʉ / nameลืม / lʉʉm / forget
หนังสือ / nǎng-sʉ̌ʉ / book

4. อุ / อู｜u / uu

短音長音
กุ / guกู / guu
สนุก / sà-nùk / funพูด / phûut / speak

Front Vowels｜前置母音

5. เกะ / เก｜e / ee

短音 เกะ 有尾音時常變成 เก็final。

เห็น / hěn / see
เกม / geem / game

6. แอะ / แอ｜ɛ / ɛɛ

แข็ง / khɛ̌ng / hard
แพง / phɛɛng / expensive

7. โอะ / โอ｜o / oo

短音 โอะ 有尾音時，母音常常o。

รถ / rót / car
หมด / mòt / finished; all gone

8. เกาะ / ออ｜ɔ / ɔɔ

ก็ / gɔ̂ɔ / then; also; well
ห้อง / hɔ̂ng / room

Compound Vowels｜複合母音

9. เกอะ / เกอ｜เออะ / เออ｜ə / əə

長音 เกอ 有尾音時，常變成 เกิ + final，例如 เดิน / dəən / walk。

เงิน / ngəən / money
เพิ่ง / phə̂ng / just recently
เริ่ม / rə̂əm / start

10. เกีย｜iia

這組在常用字裡基本當作長複合母音 iia 來學；短式 เอียะ / iaʔ 很少見，多半不是初學常用核心。

เสีย / sǐia / lose; broken
เขียน / khǐian / write

11. เกือ｜ʉa

เสื้อ / sʉ̂a / shirt
เลือก / lʉ̂ak / choose

12. อัว｜ua

無尾音常寫 อัว；有尾音時常變成 ว + final。

สวน / sǔan / garden
ปวด / bpùat / ache; hurt
รวม / ruam / include; gather

Special Vowels｜特殊與高頻母音

14. เกา｜ao

15. อาว｜aaw

อาว 是長 aa 加尾音 w，所以 Romanization 標成 aaw。

เขา / khǎo / he; she 和 ขาว / khǎaw / white 是不同母音型。

16. อำ｜am

大多數 ำ 可寫 am，但 น้ำ 作為高頻詞固定寫 náam。

Final Sound Changes｜母音遇到尾音後怎麼變

Final Sounds｜泰文尾音不是看字母原本怎麼唸

泰文很多子音放在字尾時，發音會合併成少數幾種尾音。閱讀時要先看「這個字母在字尾時屬於哪一組」，不要直接照初音念。

例如：ก / ข / ค / ฆ 放字尾時都收成 -k；ด / ต / ถ / ท / ธ / ศ / ษ / ส / จ / ช 放字尾時多收成 -t。

1. Final -k｜ก ข ค ฆ

ก ข ค ฆ 在字尾時通常收成 -k。

2. Final -t｜ด ต ถ ท ธ ศ ษ ส จ ช

這一大組放字尾時，常收成 -t。

3. Final -p｜บ ป พ ฟ ภ

บ ป พ ฟ ภ 在字尾時多收成 -p。

4. Final -n｜น ณ ญ ร ล ฬ

這組字尾常收成 -n。

5. Final -m｜ม

ม 在字尾時是 -m。

6. Final -ng｜ง

ง 在字尾時是 -ng。

7. Final -y｜ย

ย 在字尾時常收成 -y 或 i-like glide。

8. Final -w｜ว

ว 在字尾時可以明確標成 -w。

Transformation Rules｜母音變形規則

Rule 1 อะ → อั + final

กะ / ga → กัน / gan

Rule 2 เอะ → เอ็ + final

เกะ / ge → เด็ก / dèk

Rule 3 แอะ → แอ็ + final

แกะ / gɛ → แข็ง / khɛ̌ng

Rule 4 โอะ + final → hidden o

คน / khon、รถ / rót、หมด / mòt

Rule 5 อือ → อื + final

มือ / mʉʉ → คืน / khʉʉn

Rule 6 เออ → เอิ + final

เจอ / jəə → เดิน / dəən、เงิน / ngəən

Rule 7 อัว → ว + final

ตัว / dtua → สวน / sùan
