Overview｜聲調不是靠猜，是一套組合規則

判斷聲調需要 4 個資訊

  1. 主子音類別：中音字母 高音字母 低音字母
  2. 有沒有聲調符號：่ ้ ๊ ๋
  3. 音節是 live / 活尾音，還是 dead / 死尾音。
  4. 如果是 dead syllable / 死尾音，還要看母音是短母音還是長母音。
順序：先找主子音，再看 tone mark，再看 live/dead，最後才查規則。

示範：ก่อน

ก + ออ + น + ่ →

ก 是中子音，น 是 live final，有 ่，所以是 low tone：gɔ̀ɔn。

Five Tones｜泰文五個聲調

Mid Mid Tone｜平聲

聲音平穩，不特別升降。

Low Low Tone｜低聲

聲音低，通常不要拉太誇張，像往下壓住。

Falling Falling Tone｜降聲

聲音有明顯下降感，很多高頻字都是 falling tone。

High High Tone｜高聲

聲音偏高，常有比較緊、短、亮的感覺。

Rising Rising Tone｜升聲

聲音從低往上走，不能只唸成高聲。

Mini Contrast｜用一組字聽差異

這組很適合練：มา / หมา / ม้า。

Chinese Tone Bridge｜中文聲調對照表

先把英文 tone 名稱換成你熟悉的中文聲調感

這裡不是說泰文聲調等於中文聲調，而是用中文聲調當「記憶橋」。真正判讀還是要看泰文規則。

平聲 = Mid tone = no mark，像穩穩平出去 一聲 = Low tone = à，往低處壓 二聲 = Falling tone = â，先高後落 三聲 = High tone = á，高、短、亮 四聲 = Rising tone = ǎ，先低後升
這裡的「一二三四聲」是你教材裡用來對照泰文聲調的學習名稱，不是普通話聲調。之後 HTML 裡會同時保留：中文聲調名稱 + English tone name + Romanization accent。
表格裡的「中音字母 / 高音字母 / 低音字母」指的是 consonant class； 「平聲 / 一聲 / 二聲 / 三聲 / 四聲」才是 tone。兩個不要混在一起。

Live / Dead Syllables｜死活音節詳細版

一句話先記住

活音節像聲音可以「延續」：長母音開音節、或尾音是 -n / -m / -ng / -y / -w。
死音節像聲音被「切斷」：短母音開音節、或尾音是 -k / -t / -p。

Live syllable｜活音節 1：長母音開音節

沒有尾音，而且母音是長音，通常算活音節。

Live syllable｜活音節 2：活尾音

尾音是鼻音或滑音時，聲音可以自然延續，所以算活音節。

Dead syllable｜死音節 1：短母音開音節

沒有尾音，但是母音是短音，通常算死音節。聲音很短、很快被切斷。

Dead syllable｜死音節 2：死尾音

尾音是塞音 -k / -t / -p 時，算死音節。母音長短會影響低子音無調號時的聲調。

1. 先判斷 live / dead；2. 如果是 dead，再看短母音或長母音。這兩步要分開。

Leading Tone｜前引字與聲調判斷

前引字會影響「主子音類別」判斷

有前引字時，不一定只看實際發音的後面子音。前面的子音可能把自己的子音類別帶進聲調判斷，尤其是 ห นำ、高音字母前引、中音字母前引。

  • ห นำ：ห 幾乎不發音，但讓後面的低音子音照高音字母規則判斷聲調，例如 หมา / mǎa、หนู / nǔu、หลาย / lǎai。
  • 高音字母前引：例如 ขนม / khà-nǒm、สนุก / sà-nùk、ถนน / thà-nǒn，前面的高音字母會影響後面音節的聲調。
  • 中音字母前引：例如 ตลาด / dtà-làat、จมูก / jà-mùuk，前面的中音字母也可能影響後面音節判斷。
  • อ นำ：อย่า / yàa、อยู่ / yùu、อย่าง / yàang、อยาก / yàak 先整組記成 y- 起音。

判斷流程加入前引字

先看是否有前引字→決定聲調用的子音類別→看 tone mark→看 live / dead→看長短母音→得到 tone
實務上：遇到 ห นำ / อย / ขนม / ตลาด 這類字，先不要用一般單一子音規則硬判，要先辨認前引結構。

Decision Flow｜聲調判讀流程

Step 1找主子音類別

ก→中音字母 ข→高音字母 ค→低音字母

主子音類別 是整個聲調判讀的第一步。沒有找 class，後面全部會亂。

補充：如果字裡有前引字，例如 ห นำ、อย、ขนม、ตลาด，請先辨認前引字，再決定聲調要用哪個子音類別判斷。

Step 2看有沒有聲調符號

如果有 ่ ้ ๊ ๋，先用「子音類別 × 聲調符號」表判斷。

Step 3沒有符號時，看 live / dead

沒有 tone mark 時，才進入 live/dead 判斷。

Step 4如果是 dead syllable，再看長短母音

長短母音尤其會影響 低子音 無調號死音節。

ร + ั + ก→
ม + า + ก→

Step 5把規則變成讀音

ร+เ-ือ+ง+่→

ร 是低子音，่ 在低子音上通常給 falling tone，所以 เรื่อง 是 rʉ̂ang。

最終整合流程

前引字 / 主子音→有無聲調符號→Live / Dead→長短母音→Tone 結果
這是 Part 4 最重要的順序。先處理前引字，才不會把 หมา、ขนม、ตลาด 這類字判錯。

Mistakes｜常見聲調錯誤

1. 以為 high consonant = high tone

錯。高子音只是 class，不代表一定高聲。

2. 只看聲調符號，不看 class

Thai子音類別MarkTone
ก่า中音字母่gàa low tone
ค่าlow tone่khâa falling

同樣是 ่，不同子音類別 結果不同。

3. 忽略 dead syllable 長短音

兩個都是低子音 + dead final，但因母音長短不同，聲調不同。

4. ห นำ 先不要硬套一般低子音規則

หนู หมา หลาย หวาน 這類字有 ห นำ，會影響後面低子音的聲調規則。

這類前引字已在本篇 Leading Tone 區整理，拼讀細節可搭配 Part 3。
