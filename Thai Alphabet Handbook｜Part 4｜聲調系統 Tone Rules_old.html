Overview｜聲調不是靠猜，是一套組合規則

判斷聲調需要 4 個資訊

  1. 主子音 class：中 高 低
  2. 有沒有聲調符號：่ ้ ๊ ๋
  3. 音節是 live 還是 dead。
  4. 如果是 dead syllable，還要看母音長短。
順序：先找主子音，再看 tone mark，再看 live/dead，最後才查規則。

示範：ก่อน

ก + ออ + น + ่ → ก่อน gɔ̀ɔn before

ก 是中子音，น 是 live final，有 ่，所以是 low tone：gɔ̀ɔn。

Five Tones｜泰文五個聲調

Mid Mid Tone｜平聲

聲音平穩，不特別升降。

Low Low Tone｜低聲

聲音低，通常不要拉太誇張，像往下壓住。

Falling Falling Tone｜降聲

聲音有明顯下降感，很多高頻字都是 falling tone。

High High Tone｜高聲

聲音偏高，常有比較緊、短、亮的感覺。

Rising Rising Tone｜升聲

聲音從低往上走，不能只唸成高聲。

Mini Contrast｜用一組字聽差異

這組很適合練：มา / หมา / ม้า。

Live / Dead Syllables｜死活音節

Live syllable｜活音節

通常包含：

  • 長母音結尾，沒有 dead final。
  • 尾音是 -n / -m / -ng / -y / -w。

Dead syllable｜死音節

通常包含：

  • 短母音結尾。
  • 尾音是 -k / -t / -p。

Final groups for tone rules

Dead-k / -t / -pรัก / rák, คิด / khít, ชอบ / chɔ̂ɔp
死活音節是聲調規則裡最容易忽略的一步。不要只看 tone mark；沒有 tone mark 的字也要靠 live/dead 判斷。

Decision Flow｜聲調判讀流程

Step 1找主子音 class

ก→Middle ข→High ค→Low

例如 ก่อน 的主子音是 ก，所以是中子音。

Step 2看有沒有聲調符號

有符號時先查 tone mark 表；沒有符號時才看 live/dead。

Step 3沒有符號時，看 live / dead

WordClassLive / DeadTone
กินmiddlelive (-n)gin mid
ขอhighlivekhɔ̌ɔ rising
คนlowlive (-n)khon mid
รักlowshort dead (-k)rák high
มากlowlong dead (-k)mâak falling

Step 4把規則變成讀音

ร+เ-ือ+ง+่→ เรื่อง rʉ̂angmatter

ร 是低子音，่ 在低子音上通常給 falling tone，所以 เรื่อง 是 rʉ̂ang。

High-Frequency Examples｜高頻字判讀

ได้

ด+ไ-+้→ได้ dâaican; get

ด 中子音 + ้ → falling tone。

ก็

ก+อ+็→ก็ gɔ̂ɔthen; also

高頻特殊短小詞，直接固定記：ก็ / gɔ̂ɔ。

ต้อง

ต+อ+ง+้→ต้อง dtɔ̂ngmust

ต 中子音 + ้ → falling tone。

เรื่อง

ร+เ-ือ+ง+่→เรื่อง rʉ̂angmatter

ร 低子音 + ่ → falling tone。

ขอ

ข+อ→ขอ khɔ̌ɔask for

ข 高子音 + live no mark → rising tone。

สอง

ส+อ+ง→สอง sɔ̌ɔngtwo

ส 高子音 + live no mark → rising tone。

ซื้อ

ซ+ือ+้→ซื้อ sʉ́ʉbuy

ซ 低子音 + ้ → high tone。

รัก

ร+ั+ก→รัก ráklove

ร 低子音 + short dead syllable → high tone。

มาก

ม+า+ก→มาก mâakvery; much

ม 低子音 + long dead syllable → falling tone。

ผิด

ผ+ิ+ด→ผิด phìtwrong

ผ 高子音 + dead syllable → low tone。

Mistakes｜常見聲調錯誤

1. 以為 high consonant = high tone

錯。高子音只是 class，不代表一定高聲。

2. 只看聲調符號，不看 class

ThaiClassMarkTone
ก่าmiddle่gàa low
ค่าlow่khâa falling

同樣是 ่，不同 class 結果不同。

3. 忽略 dead syllable 長短音

兩個都是低子音 + dead final，但因母音長短不同，聲調不同。

4. ห นำ 先不要硬套一般低子音規則

หนู หมา หลาย หวาน 這類字有 ห นำ，會影響後面低子音的聲調規則。

這會在 Part 5 或 Appendix「ห นำ / อักษรนำ」中詳細整理。

1. 判斷 tone

  5. มาก

ขอ / khɔ̌ɔ / rising tone

รัก / rák / high tone

มาก / mâak / falling tone

2. 選出 live / dead

1. กิน / 2. คิด / 3. เมือง / 4. ชอบ / 5. แล้ว

Answer

กิน / gin / live (-n)

คิด / khít / dead (-t)

เมือง / mʉang / live (-ng)

ชอบ / chɔ̂ɔp / dead (-p)

แล้ว / lɛ́ɛw / live (-w)
