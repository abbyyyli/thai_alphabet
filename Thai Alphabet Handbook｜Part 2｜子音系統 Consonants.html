Overview｜先懂 44 個子音的真正用途

1. 子音不只是發音，也決定聲調

泰文子音分成三類： 中 高 低 。之後學聲調時，必須用「子音類別 + 長短母音 + 死活音節 + 聲調符號」一起判斷。

ก กา ก่า ก้า ก๊า ก๋า
gaa / gàa / gâa / gáa / gǎa
The consonant class helps determine the tone pattern.

2. 閱讀時先抓四件事

  1. 開頭子音是誰？例如 ก / g
  2. 它是中、高、低哪一類？
  3. 母音是短音還是長音？
  4. 尾音是活音還是死音？例如 -n / -ng 活；-k / -t / -p 死。

初音和尾音最容易混的字

  • ก initial g，final -k
  • ด initial d，final -t
  • บ initial b，final -p
  • ร ล ฬ initial r / l，final -n

Reading rhythm｜尾音不爆破

泰文的 final -k / -t / -p 多半是「收住」而不是像英文一樣明顯爆破。

Tone Notes｜先記聲調規則前的關鍵

1. Class 不等於聲調本身

高子音不是「一定唸高聲」，低子音也不是「一定唸低聲」。它們只是用來計算聲調的分類。

ขา / khǎa / leg
คา / khaa / stuck; pending
兩個都是 kh initial，但 class 不同。

3. 為什麼需要記孤低子音

低子音裡有一群沒有直接對應高子音的字，常稱為「孤低子音」。未來學 ห นำ 時會很重要。

常見例子：หมา / mǎa / dog， 因為 ห 帶著後面的低子音進入高子音聲調規則。這會在 ห นำ 篇詳細整理。

Practice｜練習

1. 找出 class

判斷以下字母是中、高、低：

ก ข ค ง จ ฉ ช ด ต ถ ท น บ ป ผ พ ฟ ม ย ร ล ว ห อ

Answer

Middle ก จ ด ต บ ป อ

High ข ฉ ถ ผ ฝ ส ห

Low ค ง ช ซ ท น พ ฟ ม ย ร ล ว

2. 初音 vs 尾音

請讀出 romanization：

  1. มาก
  2. คิด
  3. ชอบ
  4. อาหาร
  5. แล้ว
Answer

มาก / mâak / very

คิด / khít / think

ชอบ / chɔ̂ɔp / like

อาหาร / aa-hǎan / food

แล้ว / lɛ́ɛw / already; then
